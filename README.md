🧬 KaryonScan

KaryonScan é uma aplicação web que utiliza inteligência artificial para identificar, organizar e analisar cromossomos a partir de imagens, auxiliando estudantes e profissionais na área de biologia, genética e saúde.

🚀 Objetivo:

O projeto foi desenvolvido com o intuito de facilitar a visualização e organização de cromossomos em um cariótipo, tornando o processo mais rápido, intuitivo e acessível para fins educacionais e laboratoriais.

  Funcionalidades:
  
  Upload de imagens contendo cromossomos
  Detecção automática dos cromossomos na imagem
  Organização dos cromossomos em pares
  Identificação de possíveis anomalias (em desenvolvimento)
  Interface visual interativa e intuitiva
  
  Tecnologias Utilizadas:
  
HTML5
CSS3
JavaScript
Inteligência Artificial para detecção de imagem
(Opcional) Bibliotecas de visão computacional

  Aplicações:

Ensino de genética e biologia
Apoio em aulas práticas
Ferramenta auxiliar para análise cromossômica
Projetos educacionais com IA
